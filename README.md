# PYTHON 알고리즘 공부

## 소개
이 리포지토리는 PYTHON을 사용한 알고리즘 학습을 목적으로 합니다. 다양한 알고리즘을 구현하고 이해하는 것을 목표로 하고 있습니다.

## 학습 목표
- 기본 적인 알고리즘 이해
- 자료 구조 활용
- 알고리즘 문제 해결 능력 향상

## 알고리즘 목록
- 버블 정렬
- 선택 정렬
- 삽입 정렬
- 힙 정렬
- 병합 정렬
- 퀵 정렬
- 기수 정렬
- 카운팅 정렬
- 이진 탐색 트리
- 해시 테이블
- 깊이 우선 탐색 (DFS)
- 너비 우선 탐색 (BFS)
- 위상 정렬 (TS)
- 크루스칼
- 프림
- 다익스트라
- 그리디
